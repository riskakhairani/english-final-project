Personal Pronouns

Personal pronouns are words that are used to replace a person or thing. These personal pronouns are further differentiated based on their behavior as objects and subjects. Personal pronouns are divided into 2, namely as follows.

Subject Pronouns
  • I, example : I go to school early.
  • You, example : You are a boy.
  • We, example : We are busy today.
  • They, example : They are students.
  • He, example : He is a Colonel.
  • She, example : She is a good girl.
  • It, example : It is a car.
Object Pronouns
  • Me, example : Please do not disturb me.
  • You, example : She wants you to stay here.
  • Us, example : Come on, join us!
  • Them, example : If you involve them in this project, I will quit.
  • Him, example : I am not sure that was him.
  • Her, example : Why do you always talk about her?

Possessive Adjective

Possessive adjective are words that are used to express possessions. They can not stand alone, so they are followed by a noun. Possessive Adjective can function as an object or subject, but remember that this possessive adjective cannot stand alone. There are several forms of possessive adjectives, namely :

  • My, example : My mother went to Monica.
  • Your, example : That is your bag?
  • Our, example : Her husband is a doctor.
  • Their, example : Their problems are serious.
  • His, example : That is his house.
  • Her, example : Her husband is a doctor.
  • Its, example : Its door is broken.
Possessive Pronouns

Possessive pronouns are words that are used to express possessions, that can stand alone. They are not followed by a noun. There are several forms of possessive pronouns, namely :

  • Mine, example : Tina is a friend of mine.
  • Yours, example : I am yours.
  • Ours, example : This classroom is ours.
  • Theirs, example : The phone in the bag is theirs.
  • His, example : The choice was his.
  • Hers, example : This clothes is hers.
  • Its, example : The cat licked its paw
Reflexive Pronouns

Reflexive pronouns are words that are used to express somethig that has a reciprocal relation. When using a reflexive pronoun, it is usually more to express an unconscious action or reflex movement. Some forms included in reflexive pronouns are :

  • Myself, example : When i was cutting bread, i cut myself.
  • Yourself, example : You will hurt yourself.
  • Ourselves, example : We pay for ourselves.
  • Themselves, example : They are pleased with themselves.
  • Himself, example : He bought himself a new motorcycle.
  • Herself, example : She hurt herself.
  • Itself, example : The cat is washing itself.
